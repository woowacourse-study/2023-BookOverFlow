## 복잡도
### 4.1.1 복잡도와 빅오 표기법
- 시간 복잡도 : 알고리즘의 실행 시간을 정량화하는 것
- 공간 복잡도 : 실행하는데 필요한 메모리 사용량을 정량화하는 것

<br><br>

- 빅오 표기법
    - 입력 값에 대한 수식에서 최고차항을 기준으로 알고리즘이 수행되는 최악의 시간 복잡도를 표한한다.
    - 최고차항을 기준으로 표현하는 이유는 연산의 수가 극한에 수렴할 때 나머지 항이 복잡도에 미치는 영향이 미미하기 때문이다.
